# Vize Rtyně

Čistá beta webu s úvodní mapou, jednotlivými místy pod sebou, porovnávacími slidery a editorem pozic bodů.

## Přidaná místa

- Žabárna
- Park před Kampeličkou
- Hornická ulice
- Náměstí
- Centrum města
- Odpadová místa

## Editor mapy

Editor otevřeš na `editor.html`. Přetáhni body, stáhni nový `places.json` a nahraj ho do složky `data/`.
