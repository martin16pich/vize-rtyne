# Vize Rtyně

Čistá verze projektu pro nový GitHub repozitář.

## Struktura

- `index.html` – veřejný web
- `editor.html` – editor pozic bodů na hlavní mapě
- `style.css` – vzhled webu
- `script.js` – navigace a body na hlavní mapě
- `slider.js` – porovnávací slider
- `hotspots.js` – body a karty u Žabárny
- `data/places.json` – pozice bodů na hlavní mapě
- `data/hotspots.json` – body u Žabárny
- `assets/` – obrázky webu

## Editor mapy

Po nahrání na Netlify otevři:

`/editor.html`

Přetáhni body na mapě a stáhni nový `places.json`. Soubor potom nahraj zpět do složky `data/`.

## Nasazení

Obsah této složky nahraj přímo do kořene nového GitHub repozitáře.
